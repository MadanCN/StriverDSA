Basic Commands:
 

ls => Shows list of files and folders in the current directory
mkdir => Creates a directory
cd => Change directory
touch => Create and Update files
git init => Initialize an empty git reprository in the current folders
ls -a => Shows hidden files
vi => opens files
cat => display file content

git status => Shows the status of files (Untracked and staged)
git add => Add the untracked files to comit/stage
git commit -m "message" => Create a snapshot of the staged changes along a timeline of a Git projects history.
git restore --staged <filename> => Unstage the staged changes
ESC :wq => To come out of screen if you use | git commit | 
git log => To see all the message logs in reprository
